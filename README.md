# eslint-config-jaypie 🐦‍⬛🧹

Linting rules for Jaypie projects

## 🎯 Goals

Jaypie ecosystem coding style, Prettier, and bespoke rules found to improve development

## 📋 Usage

### Installation

```bash
npm install --save-dev eslint-config-jaypie
```

### Configuration

```javascript
// .eslintrc.js
module.exports = {
  extends: ["jaypie"],
};
```
